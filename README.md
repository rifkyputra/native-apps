# Native Apps


## Getting Started

Jakarta Suite is a flutter project that create apps to use system settings from native platform

## Target Platform

- Windows
- Android
- ios
- macOS
- web
- Linux (Gnome & KDE)

## Apps 
- Calculator
- Converter
- Weather
